# taesync_gui
阿里百川多媒体处理图形化GUI上传工具

##工具说明：<br>  
阿里百川的图形化GUI<br>  
省去了手动编写配置文件、命令行运行的痛苦。<br>  
图形化一键保存、运行上传。<br>  

##使用方法：<br>  
将此文件放到和taesync.exe同目录下，运行即可。<br>  

##已知问题：<br>  
如果同目录不存在百川的默认config文件夹，将会导致运行时错误。<br>  

##未来开发计划：<br>  
以Tab Control的形式提供七牛云、又拍云的配置，跟进官方命令行工具的开发进度。<br>
