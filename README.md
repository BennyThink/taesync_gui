# taesync_gui
阿里百川多媒体处理图形化GUI上传工具

##工具说明：<br>  
阿里百川的图形化GUI，全新1.2版本<br>  
省去了手动编写配置文件、命令行运行的痛苦。<br>  
图形化一键保存、运行上传。<br>  

##使用方法：<br>  
将此文件放到和taesync.exe同目录下，运行即可。<br>
